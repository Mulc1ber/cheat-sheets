# Методы встроенного класса `Math`

Описание

## Список методов `Math`:

### `Math.abs(x)`:

Возвращает абсолютное значение числа x.

### `Math.acos(x)`:

Возвращает арккосинус числа x в радианах.

### `Math.acosh(x)`:

Возвращает гиперболический арккосинус числа x.

### `Math.asin(x)`:

Возвращает арксинус числа x в радианах.

### `Math.asinh(x)`:

Возвращает гиперболический арксинус числа x.

### `Math.atan(x)`:

Возвращает арктангенс числа x в радианах.

### `Math.atanh(x)`:

Возвращает гиперболический арктангенс числа x.

### `Math.atan2(y, x)`:

Возвращает арктангенс частного y и x.

### `Math.cbrt(x)`:

Возвращает кубический корень числа x.

### `Math.ceil(x)`:

Округляет число x вверх до ближайшего целого.

### `Math.clz32(x)`:

Возвращает количество ведущих нулевых битов в 32-битном двоичном представлении числа x.

### `Math.cos(x)`:

Возвращает косинус числа x (в радианах).

### `Math.cosh(x)`:

Возвращает гиперболический косинус числа x.

### `Math.exp(x)`:

Возвращает e в степени x, где e — основание натуральных логарифмов.

### `Math.expm1(x)`:

Возвращает e^x - 1.

### `Math.floor(x)`:

Округляет число x вниз до ближайшего целого.

### `Math.fround(x)`:

Возвращает ближайшее число с плавающей запятой одинарной точности, представляющее x.

### `Math.hypot([x[, y[, …]]])`:

Возвращает квадратный корень суммы квадратов аргументов.

### `Math.imul(x, y)`:

Возвращает результат C-like 32-битного умножения аргументов x и y.

### `Math.log(x)`:

Возвращает натуральный логарифм (по основанию e) числа x.

### `Math.log1p(x)`:

Возвращает натуральный логарифм (по основанию e) от 1 + x.

### `Math.log10(x)`:

Возвращает логарифм числа x по основанию 10.

### `Math.log2(x)`:

Возвращает логарифм числа x по основанию 2.

### `Math.max([x[, y[, …]]])`:

Возвращает наибольшее из нуля или более чисел.

### `Math.min([x[, y[, …]]])`:

Возвращает наименьшее из нуля или более чисел.

### `Math.pow(x, y)`:

Возвращает x возведенное в степень y.

### `Math.random()`:

Возвращает псевдослучайное число между 0 (включительно) и 1 (не включая 1).

### `Math.round(x)`:

Округляет число x до ближайшего целого.

### `Math.sign(x)`:

Возвращает знак числа x, указывая, является ли x положительным, отрицательным или нулем.

### `Math.sin(x)`:

Возвращает синус числа x (в радианах).

### `Math.sinh(x)`:

Возвращает гиперболический синус числа x.

### `Math.sqrt(x)`:

Возвращает положительный квадратный корень числа x.

### `Math.tan(x)`:

Возвращает тангенс числа x (в радианах).

### `Math.tanh(x)`:

Возвращает гиперболический тангенс числа x.

### `Math.toSource()`:

Возвращает строку, представляющую исходный код объекта Math. Этот метод не стандартизирован и не рекомендуется к использованию.

### `Math.trunc(x)`:

Возвращает целую часть числа x, удаляя любые дробные цифры.
